# Appendix C: Glossary

- SOF: Start of Frame marker bytes.
- CRC-16-CCITT: Error detection polynomial (0x1021) with init 0xFFFF.
- STATUS/DATA/COMMAND/ACK/ERROR: Protocol message types.
- ActiveMap/HealthMap: Bitmaps of sensor state and health.
- `DeviceClient`: Host API facade over connection and parser.
- `SerialPortConnection`: Transport abstraction for serial/TCP/virtual.
